# 🎣 CodeAlpha — Phishing Awareness Training (Task 2)

An interactive web-based phishing awareness training module with 5 sections and a final quiz.

## Modules
1. **What Is Phishing?** — Types (Email, SMS, Voice, Spear)
2. **Spot a Phishing Email** — Annotated real-world fake email with red flags
3. **Social Engineering Tactics** — Interactive accordion (Urgency, Authority, Fear, Greed, Pretexting)
4. **Best Practices** — 8 actionable defense tips
5. **Final Quiz** — 5 questions with instant feedback and score

## How to Run
Just open `index.html` in any browser — no server needed.

## Tech
- Pure HTML/CSS/JS — no dependencies
- Fully responsive
